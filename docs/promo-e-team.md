我知道你不想装新东西。你现在的日子过得挺好，ChatGPT 也能用，VS Code 也够用，没必要折腾。

但就花 5 分钟试试，不好用直接删。

---

**Sman 是什么？** 一个桌面应用，打开选你的项目目录，就能跟 AI 对话。和 ChatGPT 的区别是它认识你的代码——自动读、自动分析、不用你手动喂上下文。

**对我来说有什么用？**

- 问它项目相关的问题，它知道上下文，不用你每次复制粘贴一堆代码
- 让它帮你写代码、写测试、写 commit message，直接在你项目里改好
- 你和它聊过的东西它会记住，下次不用重复说
- 它记的东西跟着 git 走，你 pull 了代码同事的 AI 也能知道你讨论过什么

**要装什么吗？** 不用。下载打开，设置里填个 API Key（公司内网的模型地址我已经配好了），选项目目录就能用。

**和 Cursor 冲突吗？** 不冲突，各干各的。

**数据去哪了？** 都在你自己机器上。内网模型，本地存储，什么都没往外传。
