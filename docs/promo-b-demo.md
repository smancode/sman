# B 版 — 效率炫耀型

> 2 分钟干完你 30 分钟的活

```
你: 加一个接口，手机号查注册状态，没注册的自动发邀请短信

Sman:
  → 分析了你项目现有的接口风格
  → 写好了代码
  → 写了三个测试用例
  → 跑了测试，全过
  → commit 已提好

2 分钟。你本来要干 30 分钟。
```

**这不是 ChatGPT 给你一段代码让你自己粘。** Sman 直接在你项目里改，测试跑通，Git 提交写好。

它认识你的项目——打开的时候选一下项目目录，它自动读代码、分析结构、加载能力。不用配置，不用写 Prompt。

还有 AUTO 模式，点一下闪电图标，AI 自己做所有决策，一路干到底。想精细控制就关掉，想快速出活就开着。

---

重复的活也行。每周的项目报告、健康检查、覆盖率统计：

```
设一个定时任务，用大白话说：
"每周一早上 9 点：统计变更 → 跑测试 → 生成报告 → 发群里"

你到工位的时候，报告已经在群里了。
```
