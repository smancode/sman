# 切换检查清单

## 切换前检查

- [ ] 所有测试通过
- [ ] 类型检查通过
- [ ] Lint 检查通过
- [ ] 构建成功
- [ ] Tauri 构建成功
- [ ] 核心流程验证通过（创建任务 -> 执行 -> 状态更新）

1. 冻结源仓 `smanclaw-desktop` 的前端功能变更
2. 锁定基线 commit
3. 更新所有开发环境的仓库指向
4. 通知团队成员

## 切换后验证

- [ ] 创建项目功能正常
- [ ] 执行任务功能正常
- [ ] 聊天功能正常
- [ ] 设置功能正常
- [ ] 任务进度显示正常

## 数据迁移（如需要）

1. 从 `~/.smanclaw-desktop/` 导出数据
2. 转换为新格式（如有变更）
3. 导入到新系统
