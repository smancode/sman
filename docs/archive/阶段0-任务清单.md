   - 流式处理 `handleStream()`
   - 错误重试

4. **ResponseParser.kt** (~250 行)
   - JSON 提取 `extractToolCalls()`
   - 多格式解析（6 种格式）
   - 容错处理

5. **ToolExecutor.kt** (~200 行)
   - 工具调用 `executeTool()`
   - 子会话隔离
   - 结果处理

   - Part 推送 `pushPart()`
   - 摘要生成 `generateSummary()`
